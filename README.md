# 💄 Beauty App

Welcome to **Beauty App** – a simple and user-friendly beauty and skincare website developed using **HTML**.

This project is designed to help users explore beauty-related content, including skincare tips, makeup tutorials, haircare guidance, beauty blogs, podcasts, and appointment booking features in one place.

---

## ✨ Features

- 🏠 Responsive Home Page
- 💆 Skincare, Makeup, Haircare & Nail Art Tutorials
- 📝 Beauty Blog Section
- 🎧 Beauty Podcast Section
- 📍 Contact Us with Google Maps Integration
- 📋 User Information Form
- 📅 Appointment Booking Section
- 🔗 External Tutorial Links
- 👩‍💻 Clean and Beginner-Friendly HTML Structure

---

## 🛠️ Technologies Used

- HTML5
- Google Maps Embed
- YouTube Video Links
